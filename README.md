# 🎄 Árvore de Natal Inteligente com Lógica Digital

Projeto desenvolvido com o objetivo de aplicar conceitos de **Eletrônica Digital** e **Circuitos Lógicos**, utilizando portas lógicas, LEDs e um visor de sete segmentos para criar uma **Árvore de Natal interativa**.

O sistema foi montado utilizando **protoboards e componentes eletrônicos discretos**, permitindo demonstrar na prática o funcionamento de **operações booleanas e circuitos combinacionais**.

---

# 📚 Objetivo do Projeto

O objetivo deste projeto é demonstrar, de forma prática e visual, a aplicação de **portas lógicas digitais** em um circuito funcional.

A Árvore de Natal utiliza combinações de **entradas lógicas através de interruptores**, que controlam o comportamento dos **LEDs e do visor de sete segmentos**, permitindo visualizar diferentes estados lógicos do circuito.

Este projeto também contribui para o desenvolvimento de habilidades em:

- Eletrônica Digital
- Montagem de circuitos em protoboard
- Lógica Booleana
- Organização de sistemas eletrônicos

---

# 🧰 Componentes Utilizados

O circuito foi construído utilizando os seguintes componentes:

### 🔌 Estrutura
- 6 Protoboards  
- Diversos fios Jumper  
- 1 Fonte de alimentação  

### 💡 Componentes Eletrônicos
- 4 LEDs
- 11 Resistores
- 1 Visor de Sete Segmentos

### 🎛️ Entradas do Sistema
- 3 Interruptores **DIP DPST X4**

---

# 🧠 Portas Lógicas Utilizadas

O circuito utiliza diversos **CIs de lógica digital**, responsáveis pelo processamento das entradas do sistema.

| Porta Lógica | CI Utilizado | Quantidade |
|---------------|--------------|------------|
| OR | 74HC32 | 8 |
| AND | 74HC08 | 4 |
| NOR (3 entradas) | 74HC27 | 1 |
| XOR | 74HC86 | 2 |
| Inversor (NOT) | 74HC04 | 4 |

Essas portas lógicas são responsáveis por **processar as combinações de entrada** e gerar diferentes respostas visuais no circuito.
